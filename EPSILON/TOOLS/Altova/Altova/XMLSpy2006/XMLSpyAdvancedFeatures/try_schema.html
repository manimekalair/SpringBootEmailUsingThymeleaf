Comprehensive Schema Editor

XMLSpy® Professional and Enterprise Editions include numerous advanced capabilities that let you build enterprise-class content models for mission-critical applications. The XMLSpy® Comprehensive Schema Editor makes it easy to view, generate, and document your XML Schema/DTD projects in important new ways.

This schema editor is highly configurable and provides controls that allow you to choose what parameters should be displayed in the schema design view and how the view should be formatted. It also provides a window that facilitates editing of identity constraints within a schema such as Name, Reference, Selector, and Field(s).






If your content model project is based on an existing XML Schema, DTD, or database schema you’re in luck. You can convert between the various XML Schema/DTD dialects at the click of a button. Furthermore, you can automatically generate an XML Schema or DTD from an existing database schema.






The auto-generation capabilities don’t end there. You can also automatically generate an XML Schema or DTD from use cases, and inversely, you can auto-generate sample XML instance documents from your XML Schema/DTD to test its functionality.

For working with XML Schemas, XMLSpy® also includes the unique Graphical Schema View, which allows you to manage the relationships and definitions in your schema in a visual manner, which removes much of the complexity associated with hand-coding XML Schemas. The XMLSpy Graphical Schema View allows you to view and edit components at a global level, and then edit the content model of individual components. Editing is as easy as using drag-and-drop functionality or an intelligent right-click menu to move elements, add new elements or attributes, change element properties, and so on.






Even when your content model is complete, the comprehensive schema editor saves you time with your project documentation. It allows you to save schema diagram snapshots, and moreover, it can automatically generate full documentation, including all diagrams, source fragments, and convenient cross-references, in HTML or Microsoft® Word®.






The XMLSpy® Comprehensive Schema Editor simplifies and speeds up schema development like nothing else can. Give the comprehensive schema editor a try today!
