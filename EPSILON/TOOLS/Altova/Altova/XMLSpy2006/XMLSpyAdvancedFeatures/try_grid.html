Enhanced Grid View

XMLSpy® Professional and Enterprise Editions include an enhanced grid view that shows the hierarchical structure of any XML-compliant document through a set of nested containers, which can be easily expanded and collapsed.

The XMLSpy® Enhanced Grid View provides exceptional visibility and control, allowing you to directly manipulate your XML document in this logical format using an advanced drag-and-drop interface and intelligent editing functions.









Only the enhanced grid view lets you condense an XML document to its fundamental components, then simply drag and drop those components within the document, effectively moving potentially thousands of lines of markup with a single action. Equally as important, the structured format allows context sensitive editing. This means that only relevant editing operations are enabled for each component, row or column. These context sensitive operations are easily selected from the right click menu or XMLSpy helper menus and help ensure valid, well formed results.

The grid view also boasts an embedded database/table view that can be selected from a button on the toolbar. You can switch between the two presentation modes at will. The database/table view automatically detects sequences of repeating elements and rearranges the presentation to show these elements in a tabular fashion, resembling a database or spreadsheet application. XMLSpy® creates this database/table view by taking the respective attributes and sub-elements of the repeating element and showing them as columns in the table. Data within the tables can be sorted in ascending or descending order at the click of a button.

For painless data exchange between applications, you can select elements in the table and simply copy and paste them directly into Microsoft® Access® or Excel®. This capability works both ways, also allowing you to copy data from any spreadsheet-like application and easily paste it into XMLSpy®.

The xmlspy 2004 Enhanced Grid View lets you visualize and edit your XML document in its most logical form. Give the enhanced grid view a try today!
