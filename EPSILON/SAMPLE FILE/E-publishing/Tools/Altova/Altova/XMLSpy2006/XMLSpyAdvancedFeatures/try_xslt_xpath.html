XSLT / XQuery Debuggers, XPath Evaluator

XMLSpy® Professional and Enterprise Editions include an XSLT 1.0/2.0 debugger, an XQuery debugger, and an XPath 1.0/2.0 evaluator to assist you in your development projects.



The XMLSpy® XSLT 1.0/2.0 Debugger lets you select from XSLT 1.0 or the new XSLT 2.0 standard, then step through the XSLT transformation process line-by-line. It features a three-panel interface that displays the input XML document, the XSLT stylesheet, and the output document as it is being built in real-time. You can set breakpoints on any node in either the XSLT stylesheet or the XML input document, inspect variables, check the call stack, analyze node-sets, and more. Debugger panels each contain several tabs supplying all the necessary debug state information.






XQuery is the impending W3C standard for querying XML documents and databases. Get an early start on XQuery development with XMLSpy® to learn the new language, which is an extension of XPath 2.0, and to fully prepare for when XQuery-enabled databases proliferate and stabilize. The XQuery debugger in XMLSpy® allows you to walk through an XQuery instruction step-by-step and to see the state of the XQuery processor in real-time, including all variables, any xpath-watch values, and a call-stack. Additionally, you can see the current context that the XQuery processor is evaluating, so that you can watch the construction of partial output and pinpoint any errors or mistakes with great precision and efficiency. With the debugger you can set breakpoints on any XQuery instruction, or on any node in the XML document you wish to query.






While the XSLT 1.0/2.0 and XQuery debuggers make it easy to troubleshoot and fix the most complicated XSLT stylesheets and XQuery statements, the XPath evaluator facilitates the testing and debugging of XPath expressions in XSLT stylesheets and XML Schema identity constraints.

The XMLSpy® XPath Evaluator is an invaluable tool for visualizing and evaluating the result of an XPath expression in real time. An XPath evaluation displays the corresponding node(s) in its result list window. Clicking on a node in the window highlights all of its components in grid or text view.

Using the XPath evaluator is extremely helpful for XSLT and XML Schema development, because it allows you to check your XPath expressions quickly, in an intuitive way. This is especially important as you come up to speed on the new XPath 2.0 standard. It can save you many hours of debugging time otherwise wasted trying to understand and track down problems with new XPath expressions, a likely source for errors in both XSLT and XML Schema applications.






The XMLSpy® XSLT 1.0/2.0 and XQuery Debugger and XPath 1.0/2.0 Evaluator speed up the trouble shooting and editing processes and help yield professional development results. Give them a try today!
