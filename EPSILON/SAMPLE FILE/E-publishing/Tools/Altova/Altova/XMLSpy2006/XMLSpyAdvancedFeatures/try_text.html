Advanced Text View

XMLSpy® Professional and Enterprise Editions include an advanced text view that makes XML documents easier to navigate and edit, thereby accelerating development projects. The XMLSpy® Advanced Text View provides code completion capabilities with context-sensitive entry helpers, line numbering, book marking, source folding, indentation guides, spell checking, and detailed find & replace functionality.

The code completion capability is designed to reduce problems with unbalanced brackets, unclosed elements, and other violations of the well-formedness principle. It will help you avoid these issues by displaying context-sensitive entry helpers that will automatically suggest a list of elements that may be inserted at a particular position in the document, complete an element, and insert all required attributes and child elements as soon as you hit the closing bracket on your keyboard. Available elements, attributes, and entities are determined by the XML Schema and are also selectable from windows on the XMLSpy desktop.






Line numbering allows for easy file navigation and correlation to XSLT error messages. A single line of text can extend indefinitely or can be word wrapped for easy reading. You can go directly to any line and character number using a button on the XMLSpy toolbar. Bookmarking lets you set points of reference in the text that you can instantly jump back to at any time. Bookmarks are displayed in the margin adjacent to the line numbers.

Source folding lets you expand and collapse portions of a file, taking advantage of the hierarchical structure of XML to present your document in a neat and orderly manner. Indentation guides show the logical links between corresponding opening and closing element tags within the file, further enhancing your document editing experience.

The advanced text view also supports multi-lingual spell checking and detailed XML-aware find & replace functionality. The spell-checker supports English (British, US, Canadian), German, Italian, Portuguese, Spanish, French, Dutch, Swedish, and other languages. It also includes separate English language medical and legal dictionaries. The find & replace functionality includes the ability to search for regular expressions using a drop-down helper menu in the find/replace dialog box. With this functionality you can quickly locate next or previous words or expressions and update your XML document on the fly.
